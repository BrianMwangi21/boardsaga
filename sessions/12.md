**Technical Notes**:
- Used wood color palette: #D4A373 (light oak), #C19A6B (medium), #8B4513 (dark), #6B3410 (darker)
- Board colors: #EEE8D3 (cream), #C19A6B (brown)
- Applied gradients extensively for subtle depth without textures
- Typography system uses CSS custom properties for easy theming
- Easter eggs use click-counting with timeouts to distinguish from normal clicks
- All animations are subtle and lightweight (scale transforms, opacity, translateY)
- Desktop-first approach as per requirements

**Design Decisions**:
- Stylized abstract wood tones with gradients instead of realistic textures for performance
- Flat colors with shadows instead of textures
- Barely-there animations for lean performance
- Interactive animations only (no ambient effects)
- Crimson Text serif for stories for classic reading experience
- Plus Jakarta Sans for UI for modern, clean look
